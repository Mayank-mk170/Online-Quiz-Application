# Online Quiz Application (Backend Only)

##  Project Overview
The **Online Quiz Application** is a backend system built using **Java Spring Boot**.
It allows users to register, take quizzes, view their scores, and track progress.
Admins can create and manage quizzes and questions.

##  Technologies Used
- Java 17
- Spring Boot 3.x
- MySQL (Database)
- JWT (Authentication)
- BCrypt (Password Encryption)
- Spring Web, Spring Security, Spring Data JPA
- Postman (for API Testing)
